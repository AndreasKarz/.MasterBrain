# Glossary

| Term | Meaning |
|---|---|
| MasterBrain | Local Obsidian-based knowledge system designed for both human navigation and AI context hydration. |
| MOC | Map of Content; a navigational note that links related notes. |
| DIGEST | Compact AI-facing summary layer. |
| RAW | Original source material; local-only by default. |
| Durable memory | Information likely to be useful across future sessions. |
| Local-only | Content allowed in the installed local vault but not in the public template repo. |
