## Principles

- Read DIGEST first.
- Update durable memory autonomously.
- Keep the public template sanitized.
- Use local-only storage for private content.
- Prefer compact context over large raw dumps.
- Use restricted tool access for cross-workspace MasterBrain updates instead of adding the vault to every project workspace.
